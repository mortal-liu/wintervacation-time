📅 GZ-Tutor-Plan | 寒假家教日程管理系统🌟 项目简介本项目是一款专门为寒假家教场景设计的轻量化进度追踪 Web 应用。通过数据驱动的架构，为学生（张广洲）及家长提供直观、透明的教学计划与进度实时监控。
开发者背景：本人是一名计算机科学专业大学生，目前在大连从事初二数学与物理家教工作。本项目旨在利用前端开发技术优化传统家教的沟通效率。
🚀 核心功能进度实时看板：顶部置顶进度条，根据已完成课时自动计算百分比，实时反馈教学进度。交互式打卡系统：点击日程卡片即可标记“已完成”状态，支持安卓设备轻微震动反馈。
本地数据持久化：基于 localStorage 实现，即使关闭网页或手机重启，打卡记录依然保留。响应式移动优先：针对 Android 设备进行深度优化，支持“添加到主屏幕”以 PWA 模式运行，获得原生 App 体验。
理科专业渲染：支持 LaTeX 风格渲染物理公式（如 $U, I, R$），确保学术严谨性。
🛠️ 技术栈框架：Tailwind CSS (Atomic CSS 架构)组件库：Flowbite (基于 Tailwind 的高颜值 UI 组件)逻辑层：Vanilla JavaScript (Data-driven 渲染逻辑)部署：GitHub Pages
📅 教学节奏逻辑系统内置了科学的教学分配逻辑：数学模块：40min (深度讲解) + 10min (休整) + 40min (巩固练习)。物理模块：50min (专题攻克)。考核周期：每 4 天进行一次阶段性小测（TEST），确保知识点日日清、周周结。
📖 如何使用对于家长：通过 Chrome 浏览器打开网址，点击菜单中的“添加到主屏幕”，即可在桌面随时查看。对于开发者：直接修改 index.html 中的 scheduleData 数组即可完成课程内容的更迭。📜 许可证本项目采用 MIT 许可证开源。
